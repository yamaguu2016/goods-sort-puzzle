# グッズソートパズル（簡易版）仕様

## 概要
- 3Dキューブ上に並んだフルーツタイルを、同じ種類3つずつ選んで消すパズル。
- すべてのタイルを消すとクリア。
- 難易度は 3x3x3 / 6x6x6 / 9x9x9 の3段階。

## 画面フロー
1. タイトル
   - 難易度選択（かんたん / ふつう / むずかしい）
   - スタートでチュートリアルへ
   - 設定でプレイヤー名変更
2. チュートリアル
   - ルール説明
   - 「プレイ」でゲーム開始
3. ゲーム
   - HUD表示、操作可能
   - すべて消すと結果画面へ
4. 結果
   - スコアと時間を表示
   - 最速タイムランキングを表示
   - 「もう一度」で再スタート

## ルール
- 任意の場所から同じフルーツを3つ選ぶと消去。
- 異なる種類を選んだ場合は選択がリセット。
- 選択数は最大3つ。
- クリア条件はタイルがすべて消えること。

## スコア・コンボ・タイム
- スコア: 消去したタイル数 × 100（3つなら +300）。
- コンボ: 消去の連続回数をカウントし、演出テキストを表示。
- タイム: ゲーム開始からの経過時間（mm:ss）。
- 最速タイム: localStorage に保存。
  - キー: goodsSortPuzzleBestTimeMs

## ランキング
- 最速タイムの上位10件を表示。
- クリア時にランキングへ記録。
- 1位は強調色で表示し、誰が最速か一瞬で分かる。
- 保存先: localStorage
  - キー: goodsSortPuzzleRanking

## プレイヤー名
- 初回 or 初クリア時に入力。
- 1〜10文字、日本語OK、絵文字NG。
- 空欄は `PLAYER` として扱う。
- 設定から変更可能。
  - キー: goodsSortPuzzlePlayerName

## 操作方法
### PC
- 左クリック: タイル選択
- 右ドラッグ: キューブ回転
- ホイール: 拡大・縮小

### スマホ/タブレット
- タップ: タイル選択
- 1本指ドラッグ: キューブ回転
- 2本指ピンチ: 拡大・縮小

## UI / UX
- HUD: スコア / 時間 / 最速 / ホームボタン
- ヒントテキスト: 「同じフルーツを3つ選んで消そう。」
- クリア時に結果パネルを表示

## ビジュアル
- 背景: グラデーション（ピンク→イエロー→ブルー）
- タイル: キューブ形状に絵文字＋色パネル
- 枠: 透明ラインで3Dフレーム表示

## 使用フルーツ
- Apple / Orange / Grape / Banana / Cherry / Lemon
- 絵文字 + 色分けで表現

## サウンド
- 消去時に簡易シンセ音を再生（Web Audio API）

## 技術
- Three.js（CDN読み込み）
- CanvasTextureで絵文字タイル生成
- 単一ページ構成（index.html / bootstrap.js / script.js / styles.css）

## 対応環境
- PC / スマホ / タブレット（WebGL対応ブラウザ）
